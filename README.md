# API проекта Yatube

## Описание
REST API для социальной сети блогов Yatube. Проект предоставляет интерфейс для публикации постов, комментирования и системы подписок на авторов. Доступ к изменению контента реализован через JWT-токены.

## Стек технологий
* Python 3
* Django
* Django REST Framework
* Djoser / SimpleJWT

## Как запустить проект локально
1. Активируйте виртуальное окружение.
2. Установите зависимости: `pip install -r requirements.txt`
3. Выполните миграции: `python manage.py migrate`
4. Запустите сервер: `python manage.py runserver`
